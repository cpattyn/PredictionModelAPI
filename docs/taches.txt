[DC]         2) Installation

                 + Récupération du projet depuis Github à déposer sur le poste client
                 + Architecture matérielle
                       2 machines:
                          - vm  -> Kubernetes -> API
                          - poste client -> Docker Compose -> Testeurs

                 + Depuis la vm:
                      Prérequis:
                            minikube installé + démarré
                            kubectl installé
                            commandes à lancer:
                                 minikube addons enable ingress
                      Installation API
                            exécution script create.sh
                            commande get service ...--url  -> IP + PORT
                            curl -X GET http://IP:5000/status
                 + Depuis le poste client
                      - prérequis
                            docker installé + démon docker démarré -> lien doc
                            docker compose
                            pré-requis connexion ssh vers vm pour pouvoir lancer une redirection de port via ssh
